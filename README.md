# MEVN Authentication

Basic Node.js / Express.js Server\
MongoDB Database\
Vue.js Frontend\
Passport Authentication using JWT and Local Strategy\

## Installation and Launch:

### Configuration
- Configure and Launch MongoDB
- Configure .env in server directory

### Server
```
cd mevn-client
npm install
npm run serve
```

### Client
```
cd mevn-server
npm install
npm run serve
```
